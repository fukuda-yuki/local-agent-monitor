# M26 Notes

- Proposal evaluation uses M25 proposal records as the only input source.
- `ready-for-human-approval` means the proposal can move to human approval review; it does not mean the proposal is accepted, implemented, prioritized, or proven effective.
- `blocked` is reserved for non-scope wording such as automated implementation, patch / diff generation, repository modification, or automatic winner selection.
- Evaluation text is sanitized and generic. It must not contain raw trace content, secrets, credentials, real user identity, patches, diffs, or file edits.
